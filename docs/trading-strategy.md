# 交易策略

---

## 一、交易哲学

**用概率思维做交易，用结构框架管理风险。**

核心原则：

- **不预测顶底**：等待结构确认后顺势跟随，不做逆势抄底/逃顶
- **风控是绝对红线**：任何单笔交易的亏损必须可控，不允许出现毁灭性损失
- **市场状态中性**：牛熊均有机会，关键在于方法论的一致性执行

---

## 二、EMA 均线体系

使用四根指数移动平均线（EMA）构成趋势判断框架：

| 均线   | 作用                  | 说明                                                                 |
| ------ | --------------------- | -------------------------------------------------------------------- |
| EMA21  | 短期趋势 & 入场参考线 | 核心均线，机构偏好 21 周期而非 20 周期，约 3% 的价差可降低被猎杀概率 |
| EMA50  | 中期趋势 & 止损参考线 | 与 EMA21 的相对位置决定趋势方向                                      |
| EMA100 | 中长期趋势            | 辅助趋势判断                                                         |
| EMA200 | 长期趋势基准          | 价格在 EMA200 之上为长期偏多，之下为长期偏空                         |

**EMA 排列定义**：

- **多头排列**：EMA21 > EMA50 > EMA100 > EMA200
- **空头排列**：EMA21 < EMA50 < EMA100 < EMA200
- **交叉/缠绕**：趋势不明，定义为震荡区间

---

## 三、价格结构判断

通过 Swing High / Swing Low 的演变确认趋势状态：

| 结构类型 | 定义                                                      |
| -------- | --------------------------------------------------------- |
| 多头结构 | 高点逐步抬高（Higher Highs）且低点逐步抬高（Higher Lows） |
| 空头结构 | 高点逐步降低（Lower Highs）且低点逐步降低（Lower Lows）   |
| 震荡结构 | 高低点横向平移，无方向性                                  |

**核心原则：结构未变，趋势未变。** 前高/前低为关键参考位，价格对这些位置的突破/跌破是判断结构转变的核心依据。

**Swing Point 检测参数**：回看窗口前后各 5 根 K 线，取最近 60 根 K 线判断结构方向。

---

## 四、趋势综合分类

结合 EMA 排列与价格结构两个维度：

| EMA 排列 | 价格结构   | 最终判定 |
| -------- | ---------- | -------- |
| 多头排列 | 高低点抬高 | **多头** |
| 多头排列 | 方向不明确 | 偏多     |
| 空头排列 | 高低点降低 | **空头** |
| 空头排列 | 方向不明确 | 偏空     |
| 其他     | 其他       | 震荡     |

---

## 五、超买/超卖信号

基于 RSI 与 Stochastic RSI 的复合信号系统，用于识别动量极值区域。

### 5.1 超卖信号

触发条件（全部满足）：

- 日线 RSI(14) < 35
- 日线 Stochastic RSI(14, 14) < 20

信号含义：市场可能接近阶段性低点，**开始关注做多机会**，但不直接作为入场依据。需等待结构确认（如价格站上 EMA21）后方可入场。

### 5.2 超买信号

触发条件（全部满足）：

- 日线 RSI(14) > 70
- 日线 Stochastic RSI(14, 14) > 80

信号含义（根据趋势方向区别处理）：

- **多头趋势中**（EMA21 > EMA50）→ **止盈参考**，提示注意获利了结，不作为追多或做空依据
- **空头趋势中**（EMA21 < EMA50）→ **做空入场参考**，确认反弹到位

## 六、入场规则

**总原则：突破回踩做多，跌破反弹做空。**

突破本身不是入场点，回踩确认才是。突破信号用于识别潜在机会，等待价格回踩 EMA21 获得支撑后再入场。

### 6.1 入场两要素

两个条件同时满足才出手，缺一不可：

1. **结构到位**：价格位于赔率有利的关键支撑/阻力区域（价格在支撑/阻力位 5% 以内）
2. **信号配合**：超买/超卖指标或回踩/反弹信号同步确认

> 止损不作为入场条件，而是出场/风控范畴。系统在生成入场建议时自动计算止损位并纳入交易计划。

### 6.2 做多入场（突破回踩）

触发条件（全部满足）：

1. 价格有效突破近期阻力位（收盘价 > 前一日阻力位）
2. 回踩至 EMA21 附近获得支撑（最低价 ≤ EMA21 × 1.01）
3. 收盘站稳 EMA21 之上（收盘价 > EMA21）
4. EMA21 > EMA50（均线多头排列确认）
5. 突破时放量（成交量 / 20 日均量 > 1.2），回踩时缩量

**信号时效**：突破后 10 个交易日内的回踩视为有效信号，超出则失效。

### 6.3 做空入场（跌破反弹）

触发条件（全部满足）：

1. 价格有效跌破近期支撑位（收盘价 < 前一日支撑位）
2. 反弹至 EMA21 附近遇阻（最高价 ≥ EMA21 × 0.99）
3. 收盘在 EMA21 之下（收盘价 < EMA21）
4. EMA21 < EMA50（均线空头排列确认）
5. 跌破时放量（成交量 / 20 日均量 > 1.2），反弹时缩量

**信号时效**：跌破后 10 个交易日内的反弹视为有效信号，超出则失效。

### 6.4 向上突破信号

触发条件（全部满足）：

- 今日收盘 > 近 20 日最高价
- 昨日收盘 ≤ 昨日的 20 日最高价（确认今日突破）
- 成交量 / 20 日均量 > 1.2

> 突破信号本身不触发入场建议，但用于：(1) 识别回踩做多的前提条件；(2) 已有持仓时作为加仓参考。

### 6.5 向下跌破信号

触发条件（全部满足）：

- 今日收盘 < 近 20 日最低价
- 昨日收盘 ≥ 昨日的 20 日最低价（确认今日跌破）
- 成交量 / 20 日均量 > 1.2

> 跌破信号本身不触发入场建议，但用于：(1) 识别反弹做空的前提条件；(2) 已有持仓时作为加仓参考。

### 6.6 不交易的情况

当趋势判定为"震荡"时，不做任何方向的交易，等待趋势明确后再介入。

## 七、出场规则

### 7.1 止损

基于结构设定止损位，优先使用近期摆动低/高点，EMA50 作为兜底：

| 方向 | 止损位计算                             | 逻辑                                           |
| ---- | -------------------------------------- | ---------------------------------------------- |
| 做多 | max(近期摆动低点 × 0.99, EMA50 × 0.99) | 取更紧的止损（更高的值），前提是止损位低于现价 |
| 做空 | min(近期摆动高点 × 1.01, EMA50 × 1.01) | 取更紧的止损（更低的值），前提是止损位高于现价 |

> 摆动低/高点来自最近 60 根 K 线中最接近现价的 Swing Low / Swing High。当无有效摆动点时，退回 EMA50 × buffer 作为止损。

止损触发后无条件执行，不做任何犹豫或等待反弹。

### 7.2 止盈

采用分批止盈策略：

| 阶段     | 条件                                       | 操作                                                 |
| -------- | ------------------------------------------ | ---------------------------------------------------- |
| 第一目标 | 价格到达近期阻力位（做多）或支撑位（做空） | 平仓 50%                                             |
| 移动止盈 | 剩余仓位使用 EMA21 作为移动止损线          | 收盘跌破 EMA21（做多）或站上 EMA21（做空）时全部平仓 |

### 7.3 时间止损

入场后 10 个交易日内价格未向预期方向显著运动（未达到第一目标的 50%），主动减仓 50% 或全部平仓。

## 八、持仓管理（两阶段建议系统）

系统根据是否有持仓自动切换建议模式：

### 8.1 无持仓 → 入场建议

| 建议     | 条件                                                             | 计划                           |
| -------- | ---------------------------------------------------------------- | ------------------------------ |
| **做多** | 结构到位 + 信号配合 + 多头趋势 + 回踩做多/超卖信号               | 入场价、止损位、目标位、盈亏比 |
| **做空** | 结构到位 + 信号配合 + 空头趋势 + 反弹做空/超买信号（空头趋势下） | 入场价、止损位、目标位、盈亏比 |
| **观望** | 条件不充分，说明具体缺少什么                                     | —                              |

### 8.2 有持仓 → 出场/加仓建议

按优先级从高到低：

| 优先级 | 建议                     | 触发条件                                            |
| ------ | ------------------------ | --------------------------------------------------- |
| 1      | **平仓（止损）**         | 现价穿过入场时记录的止损位                          |
| 2      | **减仓（第一目标止盈）** | 现价到达阻力位（做多）/ 支撑位（做空），建议平 50%  |
| 3      | **减仓（移动止盈）**     | 盈利中 + 收盘跌破 EMA21（做多）/ 站上 EMA21（做空） |
| 4      | **减仓（时间止损）**     | 持仓超 10 交易日 + 未达目标 50%                     |
| 5      | **加仓（回踩均线）**     | 盈利中 + 趋势完好 + 价格回踩 EMA21 附近 + 回踩信号  |
| 6      | **加仓（突破关键位）**   | 盈利中 + 趋势完好 + 突破信号                        |
| 7      | **继续持有**             | 以上均未触发，显示当前盈亏和 EMA21 移动止盈参考线   |

## 九、多周期分析框架

| 周期   | 用途                       | 关注指标                                  |
| ------ | -------------------------- | ----------------------------------------- |
| 周线   | 判断大趋势方向和周期位置   | EMA200 位置、超买/超卖信号                |
| 日线   | 制定交易计划、识别结构转变 | EMA21/50/100/200 排列、价格结构、入场信号 |
| 4 小时 | 精细化入场和出场时机       | EMA21 回踩位置、日内突破确认              |

**周期一致性原则**：大周期定方向，小周期找入场。若周线为空头结构，日线做多信号的置信度下调。

## 十、参数汇总

| 参数                | 默认值           | 说明                                  |
| ------------------- | ---------------- | ------------------------------------- |
| EMA 周期            | 21, 50, 100, 200 | 四级均线体系                          |
| RSI 周期            | 14               | 标准 RSI 周期                         |
| Stochastic RSI 周期 | 14               | 标准 StochRSI 周期                    |
| 超卖 RSI 阈值       | 35               | RSI 低于此值触发超卖                  |
| 超买 RSI 阈值       | 70               | RSI 高于此值触发超买                  |
| 超卖 StochRSI 阈值  | 20               | StochRSI 低于此值触发超卖             |
| 超买 StochRSI 阈值  | 80               | StochRSI 高于此值触发超买             |
| 突破回看窗口        | 20 日            | 近期高低点的回看范围                  |
| 放量阈值            | 1.2 倍           | 成交量 / 20 日均量                    |
| 回踩容忍度          | 1%               | 价格接近 EMA21 的判定范围             |
| 止损缓冲            | 1%               | 摆动点/EMA50 外侧缓冲                 |
| 结构到位容忍度      | 5%               | 价格在支撑/阻力位 5% 以内算结构到位   |
| Swing Point 回看    | 前后各 5 根 K 线 | Swing High/Low 检测窗口               |
| 结构判断窗口        | 60 根 K 线       | 用于判断价格结构的范围                |
| 信号时效            | 10 个交易日      | 突破/跌破后回踩信号的有效期           |
| 时间止损天数        | 10 个交易日      | 持仓超过此天数触发时间止损检查        |
| 目标完成阈值        | 50%              | 时间止损时判断是否达到目标的百分比    |
| 加仓 EMA21 容忍度   | 2%               | 价格在 EMA21 附近多少范围内算回踩到位 |

> 以上参数为初始值，需在实盘中通过回测验证后逐步调整。

## 十一、系统量化边界

**可量化（系统自动处理）**：

- EMA 计算与排列判断
- Swing Point 识别与结构分类
- RSI / StochRSI 信号触发
- 突破/跌破判断（价格 + 成交量）
- 入场信号（突破回踩 / 跌破反弹）
- 动态止损位 / 止盈位计算
- 两阶段建议（入场/出场自动切换）
- 持仓管理（建仓/平仓/加仓/减仓追踪）

**需主观判断（系统提供提示，用户决策）**：

- 宏观事件对信号的影响
- 多周期信号冲突时的取舍
- 市场极端情绪下的策略调整
