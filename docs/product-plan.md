# 产品方案：交易辅助系统

---

## 一、产品定位

面向个人投资者的交易辅助工具，将交易策略（详见 [01-trading-strategy.md](./01-trading-strategy.md)）中可量化的部分自动化，辅助用户做出更有纪律的交易决策。

**产品边界**：

- 提供信号扫描、图表展示、决策摘要
- 不做自动下单，决策权和执行权归用户
- 不做回测优化平台（后续版本可拓展）
- 不做收费服务或社交功能

---

## 二、目标用户

有一定交易经验、希望通过工具系统化执行结构化交易方法论的个人投资者。

**核心痛点**：

| 痛点                                     | 系统解决方式                        |
| ---------------------------------------- | ----------------------------------- |
| 手动盯盘效率低，容易遗漏信号             | 批量扫描自选股，自动识别信号        |
| 多标的并行关注时，难以快速判断各标的状态 | 信号扫描面板一览全局                |
| 交易纪律依赖意志力，易受情绪干扰         | 入场三要素指示灯 + 策略规则常驻提醒 |
| EMA 排列、结构判断、超买超卖等计算繁琐   | 策略引擎自动计算并可视化            |

---

## 三、核心功能

### 3.1 信号扫描面板（Signal Scanner）

对自选股列表批量扫描，输出每只标的的当前状态和活跃信号。

**每行展示字段**：

| 字段     | 说明                                                            |
| -------- | --------------------------------------------------------------- |
| 股票代码 | 标的标识                                                        |
| 当前价格 | 最新收盘价                                                      |
| 日涨跌幅 | 百分比                                                          |
| 趋势状态 | 多头 / 偏多 / 震荡 / 偏空 / 空头                                |
| 活跃信号 | 突破回踩做多 / 跌破反弹做空 / 超卖 / 超买 / 向上突破 / 向下跌破 |
| 综合评分 | 0-100 辅助参考值                                                |
| RSI      | 当前 RSI(14) 数值                                               |

**交互操作**：

- 点击任意行 → 右侧展开该标的详细图表
- 支持手动添加/删除自选股
- 一键刷新全部数据

### 3.2 详细图表（Chart View）

展示单只标的的完整技术分析视图，包含三个子图。

**主图（K 线 + 均线 + 信号标记）**：

| 元素            | 样式                  |
| --------------- | --------------------- |
| K 线            | 标准 OHLC 蜡烛图      |
| EMA21           | 橙色线                |
| EMA50           | 蓝色线                |
| EMA100          | 紫色线                |
| EMA200          | 红色线                |
| 超卖信号        | 绿色三角 ↑            |
| 超买信号        | 红色三角 ↓            |
| 突破回踩做多    | 绿色星星 + "做多"标签 |
| 跌破反弹做空    | 红色星星 + "做空"标签 |
| 阻力位 / 支撑位 | 水平虚线              |

**副图 1 — RSI(14)**：

- RSI 曲线 + 70 超买线 / 30 超卖线
- 超买超卖区域高亮

**副图 2 — 成交量**：

- 量柱（涨红跌绿）
- 20 日均量线

**时间周期切换**：3 个月 / 6 个月 / 1 年 / 2 年

### 3.3 信号摘要卡片（Signal Summary）

在图表上方展示当前标的的结构化决策摘要，包含以下模块：

**基础信息**：

- 股票代码、当前价格、涨跌幅
- 趋势标签（带颜色标识）
- 四根 EMA 当前值、RSI 当前值

**关键价位**：

- 近期支撑位 / 阻力位
- 做多止损参考位 / 做空止损参考位

**入场三要素检查**：

三个独立指示灯，全部为绿色时方可考虑入场：

| 要素     | 含义                   |
| -------- | ---------------------- |
| 结构到位 | 价格位于有利的结构位置 |
| 信号配合 | 存在活跃的入场信号     |
| 止损清晰 | 止损位距当前价合理     |

**赔率参考**：

- 风险距离：当前价到止损位的百分比
- 收益距离：当前价到近期阻力/支撑的百分比
- 盈亏比估算

### 3.4 策略规则面板（Strategy Rules）

常驻于界面左侧的策略规则提醒面板，内容为静态文本：

- 做多入场条件
- 做空入场条件
- 超卖/超买信号含义
- 出场规则要点

用于在交易过程中持续提醒用户遵守纪律，对抗情绪化操作。

### 3.5 事件日历提醒（V2）

标注影响技术信号有效性的重大事件，在信号摘要卡片中以风险提示形式展示：

| 事件类型      | 数据来源                  |
| ------------- | ------------------------- |
| 财报发布日    | Yahoo Finance / SEC Edgar |
| FOMC 利率决议 | 美联储官网                |
| 非农就业数据  | 经济日历 API              |

## 四、用户工作流

| 阶段       | 操作                         | 系统支持                    |
| ---------- | ---------------------------- | --------------------------- |
| 开盘前     | 打开仪表盘，点击"扫描全部"   | 信号扫描面板输出全局状态    |
| 发现信号   | 关注有活跃信号的标的         | 点击进入详细图表            |
| 验证三要素 | 确认结构、信号、止损是否对齐 | 摘要卡片三要素指示灯        |
| 决策执行   | 在券商平台自行下单           | 系统提供止损位和盈亏比参考  |
| 盘中监控   | 刷新查看新信号或超买提醒     | 手动刷新（V2 支持自动刷新） |
| 收盘复盘   | 对照信号与实际走势           | 图表历史回顾                |

---

## 五、默认自选股

MVP 默认覆盖主流美股：

SPY、QQQ、AAPL、MSFT、NVDA、META、TSLA、AMZN、GOOGL

用户可自行增删。

## 六、版本规划

### MVP

| 项目     | 方案                                          |
| -------- | --------------------------------------------- |
| 部署形式 | 本地 Web 应用                                 |
| 核心功能 | 信号扫描 + K 线图表 + 信号摘要 + 策略规则面板 |
| 数据源   | Yahoo Finance（免费，日线数据）               |
| 数据刷新 | 手动刷新                                      |
| 时间周期 | 仅日线                                        |

### V2

- 自动定时扫描 + 推送通知（Telegram / 邮件）
- 多周期分析（4 小时 / 周线）
- 加密货币支持（BTC / ETH，数据源 Binance API）
- 事件日历提醒（财报、FOMC、非农）
- 历史信号回测面板
- 交易日志与绩效跟踪
- 信号参数自定义

### V3

- AI 辅助分析（结合 LLM 解读财报/新闻对信号的影响）
- 多账户跟踪
